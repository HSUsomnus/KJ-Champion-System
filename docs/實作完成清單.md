## 🚀 下一步（使用者操作）

### 必要步驟

1. **建立 Supabase 專案**
   - 前往 https://supabase.com
   - 取得 `DATABASE_URL`（Transaction Mode）
   - 執行 `database/schema.sql`

2. **設定環境變數**
   - 複製 `env.example` → `.env`
   - 填入所有必要變數

3. **遷移資料**
   ```bash
   npm run migrate:members
   npm run sync:calendar
   ```

4. **部署至 Vercel**
   - 連結 GitHub Repo
   - 設定環境變數
   - 部署

5. **註冊 Calendar Watch**
   ```bash
   npm run watch:register
   ```

### 選用步驟

- 調整 Cron 頻率（`vercel.json`）
- 設定 `CRON_SECRET` 保護 Cron endpoints
- 自訂同步範圍（`calendarSyncService.js`）
