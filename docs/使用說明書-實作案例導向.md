# LINE LIFF 行事曆 — 使用說明書（實作案例導向）

> 本說明書以**實際使用案例**為主，用真實範例帶出操作方式與命名規則，方便團隊照著做。

---

## 一、項目名稱怎麼寫（必讀）

行事曆上的**每一個行程**都有一個「項目名稱」。為了一目了然、方便排程，請依下列格式填寫：

### 1.1 格式說明

| 部分 | 說明 | 範例 |
|------|------|------|
| **日期** | 可寫成 `MM/DD` 或 `MMDD` | `02/15`、`0218` |
| **時間** | **兩位數**，表示「幾點」；直接寫在項目名稱**最前面** | `14` = 14:00，`19` = 19:00 |
| **項目名稱** | 時間後面接的內容：對象 + 類型（課程／諮詢／活動等） | 小陞財務諮詢、小政金流課 |

**重點：項目名稱開頭的兩位數 = 當天幾點。** 例如 `14小陞財務諮詢` 的 **14** 就是 14:00。

### 1.2 實作案例（照這樣寫就對了）

以下都是**正確寫法**的實際例子，可直接當範本使用：

| 案例 | 項目名稱範例 | 說明 |
|------|----------------|------|
| **案例 A** | `02/15 14小陞財務諮詢(簽約諮詢)` | 2/15，下午 2 點，小陞的財務諮詢（簽約諮詢類型） |
| **案例 B** | `0218 小政金流課` | 2/18，小政的金流課（若無寫數字，時間在新增行程時再選即可） |
| **案例 C** | `02/08 19小琦保單諮詢` | 2/8，晚上 7 點，小琦的保單諮詢 |

- **14**、**19** 放在名稱前 = 當日 **14:00**、**19:00**。
- 日期可寫在標題裡（如 `02/15`）或只靠系統的「日期」欄位；名稱裡寫日期是為了**一眼看出哪一天**，方便辨識。

## 二、依情境的實作案例

### 2.1 新增一筆「學員上課」行程

**情境：** 2/18 要排「小政金流課」，下午 2 點開始。

1. 點 **「＋」或「新增行程」**。
2. **日期** 選 2/18。
3. **時間** 選 14:00（若介面有時間欄位）。
4. **類型** 選「學員上課」。
5. **標題／項目名稱** 填：`0218 14小政金流課`（或 `02/18 14小政金流課`）。
6. 儲存。

**實作範例：**  
`0218 14小政金流課` → 2/18、14:00、學員上課。

---

### 2.2 新增一筆「諮詢簽約」行程

**情境：** 2/15 下午 2 點，小陞的財務諮詢（簽約諮詢）。

1. 新增行程 → 日期選 2/15，時間選 14:00。
2. **類型** 選「諮詢簽約」。
3. **標題／項目名稱** 填：`02/15 14小陞財務諮詢(簽約諮詢)`。
4. 儲存。

**實作範例：**  
`02/15 14小陞財務諮詢(簽約諮詢)` → 2/15、14:00、諮詢簽約。

---

### 2.3 新增一筆「晚上諮詢」行程

**情境：** 2/8 晚上 7 點，小琦保單諮詢。

1. 新增行程 → 日期選 2/8，時間選 19:00。
2. **類型** 選「諮詢簽約」（或依你們的類型設定）。
3. **標題／項目名稱** 填：`02/08 19小琦保單諮詢`。
4. 儲存。

**實作範例：**  
`02/08 19小琦保單諮詢` → 2/8、19:00。

---

### 2.4 在列表裡找「某一天的行程」

**情境：** 想確認 2/15 有哪些行程。

1. 進入 **「列表」** 頁。
2. 切換分頁：**學員上課 / 活動 / 諮詢簽約**（依類型選）。
3. 選 **月份** 為 2 月。
4. 列表上會看到該類型、該月份的行程；項目名稱若含 `02/15` 或 `0215`，可快速對應到 2/15。

**實作要點：** 項目名稱前加日期（如 `02/15`、`0218`），在列表檢視時較好辨識。

---

### 2.5 分享行程給夥伴

**情境：** 要把「2/18 小政金流課」分享到 LINE。

1. 在 **行事曆** 或 **列表** 點進該筆行程（例如 `0218 14小政金流課`）。
2. 點 **「分享」**。
3. 在 LINE 內選擇要分享的聊天室或好友，送出。

**實作要點：** 項目名稱寫清楚（日期 + 時間 + 內容），對方收到分享時一眼就懂。

## 三、項目名稱速查（時間對照）

| 名稱開頭數字 | 代表時間 |
|--------------|----------|
| 09 | 09:00 |
| 10 | 10:00 |
| 14 | 14:00 |
| 19 | 19:00 |
| 20 | 20:00 |

**記法：** 項目名稱最前面的**兩位數** = 當天**幾點**（24 小時制）。例如 `14XXX` 就是 14:00 的行程。

## 五、小結

- **項目名稱**建議包含：**日期（可選） + 時間（兩位數，選填） + 內容**。
- **開頭兩位數 = 當天幾點**（如 14 = 14:00、19 = 19:00）。
- 照上述實作案例填寫，行事曆與列表會一致、好辨識，分享給夥伴也清楚。

若日後有新增類型或新範例，可再補進本說明書的「實作案例」一節。
