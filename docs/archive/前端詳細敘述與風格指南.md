# LINE LIFF 行事曆系統 — 前端詳細敘述與風格指南

> 本文件詳細說明此專案的前端架構、頁面組成、視覺風格與設計原則，供開發與維護參考。

---

## 一、概述

本專案為 **LINE LIFF（LINE Front-end Framework）** 應用程式，以手機優先（Mobile First）設計，主要透過 LINE App 內建的瀏覽器執行。前端採用 **原生 HTML / CSS / JavaScript** 開發，無使用 React、Vue 等框架，以輕量、穩定為目標。

### 1.1 技術棧

| 類型 | 技術 |
|------|------|
| 標記語言 | HTML5 |
| 樣式 | CSS3（含 CSS 變數、Flexbox、Grid） |
| 腳本 | 原生 JavaScript（ES5+） |
| 整合 | LINE LIFF SDK v2 |
| 除錯 | Eruda（可選，網址帶 `?eruda=1` 載入） |

### 1.2 檔案結構

```
public/
├── css/
│   └── style.css           # 全域樣式（單一檔案）
```

## 二、視覺風格與設計系統

### 2.1 色彩系統（CSS 變數）

所有主要色彩定義於 `:root`，便於整體調整：

| 變數名稱 | 色碼 | 用途 |
|----------|------|------|
| `--primary-color` | `#06C755` | LINE 品牌綠、主按鈕、強調色 |
| `--secondary-color` | `#00B900` | 主色深綠、hover 狀態 |
| `--text-color` | `#333333` | 主要文字 |
| `--text-light` | `#666666` | 次要文字 |
| `--bg-color` | `#F5F5F5` | 頁面背景 |
| `--card-bg` | `#FFFFFF` | 卡片、按鈕背景 |
| `--border-color` | `#E0E0E0` | 邊框、分隔線 |
| `--shadow` | `0 2px 8px rgba(0,0,0,0.1)` | 一般陰影 |
| `--shadow-hover` | `0 4px 12px rgba(0,0,0,0.15)` | 懸停陰影 |

**行程類型專用色：**

| 類型 | 背景 | 文字／邊框 |
|------|------|------------|
| 學員上課 | `#FFF9C4` | `#F57F17` |
| 活動 | `#FFEBEE` | `#C62828` |
| 諮詢簽約 | `#E8F5E9` | `#2E7D32` |

**星等專用色：**

| 星等 | 變數 | 色系 |
|------|------|------|
| 白星 | `--star-white` | 灰階 |
| 綠星 | `--star-green` | 綠色 |
| 橙星 | `--star-orange` | 橘色 |
| 紅星 | `--star-red` | 紅色 |
| 紫星 | `--star-purple` | 紫色 |

### 2.2 字型

- **系統字型堆疊**：`-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Microsoft JhengHei', '微軟正黑體', Arial, sans-serif`
- 優先使用系統內建字型，確保中文顯示良好。

### 2.3 圓角與間距

| 元素 | 圓角 | 間距 |
|------|------|------|
| 按鈕 | 8px | padding 12px 24px |
| 卡片 | 12px | padding 16px、margin-bottom 12–16px |
| 表單輸入框 | 8px | padding 12px |
| 成員／行程類型標籤 | 12px（pill 型） | padding 4px 12px |
| 導航項目 | 無（底部導航） | padding 2px 12px |

### 2.4 標題層級

| 層級 | 字級 | 字重 |
|------|------|------|
| h1 | 24px | 700 |
| h2 | 20px | 600 |
| h3 | 18px | 600 |
| 頁面標題（.page-title） | 18px | 700 |

### 2.5 按鈕樣式

- **主按鈕（.btn-primary）**：LINE 綠背景，白字，hover 時略微上浮並加深陰影
- **次按鈕（.btn-secondary）**：白底灰框，hover 變淺灰背景
- **危險按鈕（.btn-danger）**：紅底白字，用於刪除等操作
- **分享按鈕（.share-btn）**：與主按鈕同風格，搭配圖示

### 2.6 卡片樣式

- 白底、12px 圓角、輕陰影
- hover 時陰影加強，部分卡片有 `translateY(-2px)` 微動效果
- 行程字卡、成員字卡均採用此風格

## 三、版面與佈局

### 3.1 手機優先與底部導航

- 針對手機螢幕優化，最大寬度 `max-width: 768px`（桌面時置中）
- 底部導航欄固定於畫面底部，高度約 52px，為內容區預留 `padding-bottom: 54px`
- 導航項目：行事曆、列表、成員、個人

### 3.2 固定標題 + 可捲動內容

各主要頁面採用「標題固定、僅內容區可捲動」的結構，避免整頁捲動造成標題消失：

| 頁面 | 固定區 | 可捲動區 |
|------|--------|----------|
| 行事曆 | 標題、月曆（50vh）、今日行程標題列 | 行程字卡 |
| 列表 | 標題、分頁列 | 行程字卡 |
| 成員 | 標題、邀請按鈕 | 成員字卡 |
| 新增行程 | 標題 | 表單 |
| 行程詳情 | 標題 | 詳情／編輯表單 |

### 3.3 行事曆頁特殊佈局

- 上半屏（50vh）：標題 + 月曆，月曆下緣固定在螢幕一半
- 下半屏：「今日行程」標題與「新增行程」按鈕同一行，下方為行程字卡捲動區
- 月曆採用 7 欄 Grid（日～六），日期格等高，選取日以框線標示（非實心填色）

## 四、頁面說明

### 4.1 主頁（index.html）— 行事曆

- **功能**：月曆瀏覽、日期選擇、當日行程顯示、新增行程入口
- **進入流程**：LIFF  Ready → 檢查成員註冊 → 未註冊導向 profile、已註冊顯示主內容
- **登入**：未登入時顯示透明毛玻璃 overlay + LINE 登入按鈕
- **載入動畫**：康九 logo + 「載入中...」，透明毛玻璃背景

### 4.2 行程列表（list.html）

- **功能**：依類型分頁（學員上課、活動、諮詢簽約）顯示行程，可分享整月、新增行程
- **分頁**：點擊切換，active 以綠色底線標示
- **月份**：每組行程上方有月份標題，右側可分享整月或新增

### 4.3 成員列表（members.html）

- **功能**：顯示所有成員卡片（頭像、姓名、星等）、邀請新夥伴
- **成員卡片**：左側頭像、右側姓名與星等標籤，點擊進入成員詳情

### 4.4 成員詳情（member-detail.html）

- **功能**：唯讀顯示成員資料（真實姓名、Email、電話、星等、課程紀錄）
- **頁面**：無上一頁按鈕，依賴 LIFF 瀏覽器內建返回

### 4.5 個人資料（profile.html）

- **功能**：顯示／編輯個人資料、註冊表單（未註冊時）
- **模式**：唯讀顯示 ↔ 編輯表單，星等與課程紀錄可編輯

### 4.6 新增行程（add-event.html）

- **功能**：表單新增行程（標題、類型、日期時間、備註）
- **類型**：學員上課、活動、諮詢簽約
- **日期／時間**：支援整日活動；非整日時使用自訂 datePicker、timePicker（滾輪式）

### 4.7 行程詳情（event-detail.html）

- **功能**：顯示行程詳情、分享、編輯、刪除
- **模式**：詳情顯示 ↔ 編輯表單

### 4.8 獨立頁（standalone）

- `event-detail-standalone.html`：分享連結用，可單獨開啟行程詳情
- `month-events-standalone.html`：分享整月行程用

## 五、共用元件與模式

### 5.1 頁面標題列（.page-header-row）

- 左側：清除快取按鈕（SVG 重新整理圖示）
- 中央：頁面標題
- 右側：無（部分頁面由 JS 動態加入按鈕）

### 5.2 資料載入動畫

- **樣式**：透明毛玻璃（`backdrop-filter: blur`）+ 康九 logo + 「載入中...」
- **動畫**：logo 輕微脈動（`data-loading-pulse`）
- **覆蓋式**：`.data-loading-overlay` 覆蓋內容區，父層需 `position: relative` 與 `min-height`

### 5.3 統一彈窗（share-dialog.js）

- 取代 `alert`、`confirm`，風格與分享確認對話框一致
- `showAppAlert(message)`：單按鈕提示
- `showAppConfirm(message, opts)`：雙按鈕確認，回傳 true/false
- 樣式：白底卡片、圓角、陰影，按鈕使用 .btn-primary / .btn-secondary

### 5.4 分享確認對話框

- 在分享前顯示「是否要分享？」之類確認
- 是／否按鈕，風格與統一彈窗相同

### 5.5 底部導航欄（.bottom-nav）

- 固定於底部，四項目：行事曆、列表、成員、個人
- 圖示以 Emoji 呈現（📅、📋、👥、👤）
- active 狀態：文字與圖示改為 primary 綠色

## 六、表單與輸入

### 6.1 表單樣式

- `.form-group`：表單區塊，margin-bottom 20px
- `.form-label`：標籤，block 顯示，字重 500
- `.form-input`、`.form-select`、`.form-textarea`：統一 padding、圓角、focus 時綠色邊框與淡綠色光暈

### 6.2 日期／時間選擇器

- **風格**：類似 Google 表單的滾輪式選擇器
- 彈窗：半透明黑底 overlay，中央白底卡片
- 滾輪：垂直捲動，選中項目放大、其他淡化
- 日期：年、月、日三欄，分隔符顯示
- 時間：時、分兩欄

### 6.3 複選框群組

- 課程紀錄等使用 `.checkbox-group`，每個項目為 `.checkbox-label`
- checkbox 使用 `accent-color: var(--primary-color)` 與主色一致

## 七、互動與動畫

### 7.1 過渡效果

- 按鈕、卡片：`transition: all 0.3s ease`
- 按鈕 hover：`transform: translateY(-2px)`、陰影加強
- 彈窗：opacity、visibility 過渡約 0.35s

### 7.2 捲動

- 隱藏原生滾動條（`scrollbar-width: none`、`::-webkit-scrollbar { display: none }`）
- 保留捲動功能，行動裝置使用 `-webkit-overflow-scrolling: touch` 提升流暢度

### 7.3 響應式斷點

- 768px 以上：`.container` 限制 max-width: 768px，內容置中

## 八、LINE 與 LIFF 整合

### 8.1 執行模式（liff.js）

- `development`：本機測試，模擬 LINE User ID，不需真實登入
- `internal`：內測，真實 LIFF + ngrok
- `production`：正式環境

### 8.2 登入流程

- 首頁：進入時檢查登入與成員註冊，未登入顯示 overlay + LINE 登入按鈕
- 非首頁：未登入時顯示 `liff-login-overlay`（LINE 綠漸層背景）
- 登入按鈕：圓角膠囊型，主色綠底白字或白底綠字

### 8.3 分享

- 行程／整月：透過 `shareTargetPicker` 或 `shareMessage` 分享
- 邀請新夥伴：可產生 Flex 字卡分享
- 非 LINE 環境：改用 Web Share API 或複製至剪貼簿

## 十、風格總結

| 面向 | 原則 |
|------|------|
| **品牌** | 主色採用 LINE 綠（#06C755），保持與 LINE 生態一致 |
| **簡潔** | 白底、輕陰影、適度圓角，避免視覺干擾 |
| **手機優先** | 觸控友善、固定標題、可捲動內容、底部導航 |
| **一致性** | 按鈕、卡片、表單、彈窗共用設計語言 |
| **載入體驗** | 透明毛玻璃 + 品牌 logo 動畫，減少等待焦慮 |
| **無障礙** | 適當 aria-label、語意化 HTML，支援鍵盤與螢幕閱讀器基本操作 |
