## 做了什麼

- 新增邀請字卡系統：LINE Bot push Flex Message 邀請卡給使用者，由使用者自行轉發
- `invite-share.html`：使用 LINE 官方 LIFF SDK + `liff.shareTargetPicker()` 分享邀請字卡
- Bot push 後導回 LINE Bot 聊天室，方便使用者快速轉發給好友/群組
- 邀請按鈕從成員列表頁移至個人頁面（profile.html）
- 多日行程支援：月曆上跨日顯示色點，列表顯示多日行程
- 新增邀請追蹤與階層式權限系統

---

## 為什麼改

- 原本的分享方式（LINE URL Scheme）無法傳 Flex Message，改用 Bot push + LIFF shareTargetPicker
- 邀請按鈕放在個人頁面更合理（邀請是個人行為）

---

## 踩過的坑

1. **LINE URL Scheme 無法轉發 Bot 訊息**：Bot 推送的訊息無法透過 URL Scheme 轉發 → 改為 LIFF shareTargetPicker
2. **LIFF isAvailable() 不存在**：改用 `liff.isApiAvailable('shareTargetPicker')`
3. **invite-share.html 需在 LINE App 內開啟**：LIFF shareTargetPicker 只能在 LINE 內使用，外部瀏覽器無效 → 加入導回 LINE 的邏輯

---

## 未來維護注意

- `invite-share.html` 使用 LINE 官方 LIFF SDK，**不能移除或改用自製介面**
- LIFF_ID 環境變數提供 invite-share.html 使用，由 `/api/line/liff-id` 路由提供
- Bot push 功能在 `server/routes/line.js` 和 `server/services/lineService.js`
