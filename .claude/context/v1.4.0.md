## 技術架構（本版現況）

### 前端（`public/`）
- 純 HTML + 原生 JS + CSS，無框架
- 身份驗證：`public/js/liff.js`（自製 LINE Login OAuth，不依賴 LIFF SDK）
- 提供 `window.LIFF` 介面（自製）和 `window.liff` 相容層

### 後端（`server/`）
- Node.js + Express.js
- `server/server.js`：主入口，靜態服務 `public/` 目錄
- 路由：`server/routes/`（auth、calendar、member、profile、line、financial）
- 服務：`server/services/`（calendarService、memberDbService、lineService 等）

### 資料庫
- Supabase（PostgreSQL）：成員資料
- Google Calendar API：行事曆 CRUD
- Google Sheets API：報表/財務

### 部署
- 後端：Google Cloud Run
- 前端（靜態）：Vercel（直接服務 `public/`）
