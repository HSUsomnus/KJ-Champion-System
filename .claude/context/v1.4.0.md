## 做了什麼

- 移除廢棄的 React 前端（`frontend/` 目錄）及過時文件
- 開放 manager 角色存取管理中心的財務頁籤（原本僅 admin）
- 行程分享（分享後／新增後）改為純文字，與行事曆頁面和列表頁面一致
- 邀請字卡按鈕 3 改為透過 `/open-external.html` 中介頁面在外部瀏覽器開啟
- 邀請字卡按鈕 3 連結改為指向 `APP_URL`（line-liff-calendar.vercel.app）

## 為什麼改

- React 前端已廢棄（改用純 HTML/JS），移除減少混淆
- manager 角色需要存取財務資訊，原本只有 admin 可以看
- 分享功能統一為純文字格式，行為一致
