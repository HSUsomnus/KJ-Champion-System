# v2.0.2 — 財力頁功能修復 (hotfix)

**日期**：2026-04-06
**分支**：hotfix/financial-upload-accept → main
**git tag**：v2.0.2

---

## 變更摘要

大型 hotfix，修復 v2.0.0 後財力頁多個功能失效問題：檔案選擇器、試算表預覽（Excel 顏色/樣式/合併儲存格）、Web Share API 流程、離開守衛。同時新增 hotfix 工作流規則與子代理開發腳本。

## 主要修復項目

| 問題 | 修復方式 |
|------|---------|
| 手機檔案選擇器無法顯示試算表格式 | accept 屬性補全 Excel MIME types |
| Excel 填色/顏色未正確顯示 | 改用 x-data-spreadsheet 渲染，補 indexed 色盤 + ARGB 解析 + theme 顏色解析 |
| 試算表欄寬/合併儲存格/欄列標題消失 | FinancialPreview 加入完整 cell 樣式支援 |
| Samsung Internet navigator.share 手勢過期 | 預先 fetch blob cache 解決 user activation 過期問題 |
| 財力選取/編輯路由混亂 | 拆為獨立路由，統一所有編輯頁 useLeaveGuard |
| useLeaveGuard beforeunload 不吃 shouldBlock 條件 | 修正條件攔截邏輯 |
| 點擊財力文件開啟方式不一致 | 改用 sheet_view_url 開啟 Google Sheets 預覽 |

## 影響範圍

- `frontend/src/pages/Financial*`（多個財力相關頁面）
- `frontend/src/hooks/useLeaveGuard.js`
- 新增 `.claude/rules/` hotfix 工作流規則
- 新增 `.claude/commands/` 子代理並行開發腳本
- 移除 post-checkout hook，改用 main merge 同步所有分支

## 架構現況（v2.0.2）

與 v2.0.0 相同（React + Vite + PWA，Cloudflare Pages + Zeabur），本版為純 bug fix，無架構變動。
