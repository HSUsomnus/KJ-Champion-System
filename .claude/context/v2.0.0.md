# v2.0.0 — React 新前端上線

**日期**：2026-04-04
**分支**：m_b_開發新UI前端 → main
**git tag**：v2.0.0

---

## 變更摘要

整個前端從純 HTML+JS（`public/`）換成 React 18 + Vite 5 + Tailwind CSS（`frontend/`）。舊 `public/` 目錄已刪除。

## 重大變更

- **前端目錄**：`public/` → `frontend/`（React SPA）
- **建構工具**：無建構 → Vite 5 + React 18
- **樣式系統**：手寫 CSS → Tailwind CSS（Warm Minimal 風格）
- **路由方式**：多 HTML 檔 → React Router SPA
- **部署方式**：Cloudflare Pages 靜態 → Cloudflare Pages + Vite build output
- **PWA**：原生 SW → Vite PWA Plugin

## UI 風格（Warm Minimal）

- 背景：`#F7F5F2`
- 強調色：`#4A7C59`
- 文字色：`#2C2C2C`
- 圓角：`rounded-xl`

## 關鍵檔案

| 檔案 | 說明 |
|------|------|
| `frontend/public/_worker.js` | Cloudflare Worker，proxy `/api/*` 至 Zeabur 後端 |
| `frontend/src/App.jsx` | React Router 主入口 |
| `frontend/src/contexts/AuthContext.jsx` | LINE Login 認證 context |
| `frontend/src/contexts/UserContext.jsx` | 使用者資料 context |
| `frontend/vite.config.js` | Vite + PWA 設定 |
| `frontend/tailwind.config.js` | Tailwind 主題設定 |

## 影響範圍

- 前端完全重建，所有頁面重新實作
- 後端無變更
- Cloudflare Pages 部署設定需更新（build command、output directory）
- `_worker.js` 已改回正式後端 URL（`kj-champion-system.zeabur.app`）
