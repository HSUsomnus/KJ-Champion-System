# v1.0.0 上下文記憶

git tag: v1.0.0
日期：2025
相關 commits：85327b5（初始提交）及後續早期開發

---

## 做了什麼

- 初始版本，純 HTML/JS/CSS 前端（`public/` 目錄）
- 月曆行事曆（`index.html` + `calendar.js`）：月視圖、日期點擊顯示當日行程
- 行程列表（`list.html`）：依類型篩選（學員上課/活動/諮詢簽約）
- 行程新增/編輯/刪除（`add-event.html`、`event-detail.html`）
- 成員管理（`members.html`、`member-detail.html`）：成員列表、星等評分、備註
- 個人資料頁面（`profile.html`）：顯示 LINE 頭像、個人進階資訊
- Supabase (PostgreSQL) 資料庫：成員資料、個人備忘
- Google Calendar API：行事曆事件 CRUD
- Google Sheets API：成員資料匯入
- Node.js + Express.js 後端
- LINE LIFF SDK 登入（後來在 v1.1.0 移除）

---

## 為什麼建立

- 康九冠軍夥伴系統：管理團體成員、行程、財務的內部工具
- 透過 LINE 帳號驗證身份，只有成員可以使用

---

## 踩過的坑

- 早期使用 React 前端（後來多次反覆，最終回到純 HTML 版本）
- LINE LIFF SDK 限制導致後來完全移除（v1.1.0）
- Google Calendar 私鑰換行問題：改用 `GOOGLE_SERVICE_ACCOUNT_JSON` 完整 JSON 字串

---

## 未來維護注意

- 行事曆類型：`學員上課`、`活動`、`諮詢簽約`（中文值，存在 Google Calendar 事件）
- Supabase 連線：`DATABASE_URL` 環境變數
- Google Calendar ID：`GROUP_CALENDAR_ID` 環境變數
- 部署目標：Google Cloud Run（後端），LINE Login OAuth callback 須在 LINE Developer Console 登記
