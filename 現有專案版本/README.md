# 現有專案版本

此資料夾對應 **current-version**（英文資料夾名，避免終端機編碼問題）。

## 📂 實際專案位置

請至專案根目錄的 **`current-version`** 資料夾操作：

```
Line_Liff/
└── current-version/   ← 實際專案內容在這裡
    ├── server/
    ├── public/
    ├── docs/
    ├── package.json
    └── ...
```

## 🚀 如何執行專案

在專案根目錄開啟終端機後執行：

```bash
cd current-version
npm install
npm run dev
```

伺服器會在 `http://localhost:8080` 啟動。

## 📝 說明

- **現有專案版本** = **current-version**（同一套專案）
- 實際開發與部署請使用 `current-version` 路徑
