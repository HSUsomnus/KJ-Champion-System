## 📦 步驟 1：安裝 Node.js

### 1.1 檢查是否已安裝

1. **開啟 PowerShell**
   - 按鍵盤 `Win + X`
   - 選擇「Windows PowerShell」或「終端機」

2. **輸入檢查命令**
   ```powershell
   node --version
   ```
   
3. **看結果**：
   - ✅ **如果顯示版本號**（例如 `v20.11.0`）→ 跳到步驟 2
   - ❌ **如果顯示錯誤** → 繼續下面的安裝步驟

### 1.2 下載 Node.js

1. **開啟瀏覽器**，前往：https://nodejs.org/

2. **下載 LTS 版本**（左邊綠色按鈕）
   - 檔案名稱類似：`node-v20.x.x-x64.msi`
   - 這是 Windows 安裝程式

3. **執行安裝程式**
   - 雙擊下載的 `.msi` 檔案
   - 點擊「Next」→「Next」→「Next」
   - **重要**：確保勾選「Automatically install the necessary tools」
   - 點擊「Install」
   - 等待安裝完成

4. **重新開啟 PowerShell**
   - 關閉舊的 PowerShell 視窗
   - 開啟新的 PowerShell

5. **再次檢查**
   ```powershell
   node --version
   npm --version
   ```
   - 兩個命令都應該顯示版本號

## 📦 步驟 3：安裝專案依賴套件

### 3.1 執行安裝命令

**確保你在專案資料夾中**，然後輸入：

```powershell
npm install
```

### 3.2 等待安裝完成

**會發生什麼？**
- 會下載很多檔案（約 150+ 個套件）
- 需要 1-5 分鐘（取決於網路速度）
- 會看到很多訊息在跑

**成功的標誌：**
```
added 150 packages, and audited 151 packages in 2m
```

**如果看到錯誤：**

❌ **錯誤：`npm ERR! code EACCES`**
- **原因**：權限不足
- **解決**：以系統管理員身分執行 PowerShell
  1. 右鍵點擊 PowerShell
  2. 選擇「以系統管理員身分執行」
  3. 重新執行 `npm install`

❌ **錯誤：`npm ERR! network`**
- **原因**：網路連線問題
- **解決**：
  1. 檢查網路連線
  2. 如果在大陸，使用淘寶鏡像：
     ```powershell
     npm config set registry https://registry.npmmirror.com
     ```
  3. 重新執行 `npm install`

### 3.3 驗證安裝

檢查 `node_modules` 資料夾是否建立：

```powershell
Test-Path node_modules
```

如果顯示 `True`，表示成功！✅

## 🔐 步驟 4：建立環境變數檔案

### 4.1 複製範例檔案

在 PowerShell 中輸入：

```powershell
Copy-Item env.example .env
```

### 4.2 開啟 .env 檔案

**方法一：使用記事本**
```powershell
notepad .env
```

**方法二：使用檔案總管**
1. 在檔案總管中找到 `.env` 檔案
2. 如果看不到，點擊「檢視」→ 勾選「隱藏的項目」
3. 右鍵 `.env` → 「開啟方式」→ 「記事本」

### 4.3 查看檔案內容

`.env` 檔案應該長這樣：

```env
# Google API 設定
GOOGLE_SERVICE_ACCOUNT_EMAIL=your-service-account@project-id.iam.gserviceaccount.com
GOOGLE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n"
GOOGLE_PROJECT_ID=your-project-id

# Google Calendar 設定
GROUP_CALENDAR_ID=your-group-calendar-id@group.calendar.google.com

# Google Sheets 設定
MEMBER_SHEET_ID=your-google-sheet-id
MEMBER_SHEET_NAME=成員資料

# LINE LIFF 設定
LIFF_ID=your-liff-id
LINE_CHANNEL_ID=your-channel-id
LINE_CHANNEL_SECRET=your-channel-secret

# 伺服器設定
PORT=8080
NODE_ENV=production
```

**⚠️ 現在先不要修改這些值！**
- 這些是範例值
- 我們會在後續步驟中取得真實的值
- 先儲存檔案（Ctrl + S）即可

## 🚀 步驟 5：測試伺服器

### 5.1 啟動開發伺服器

在 PowerShell 中輸入：

```powershell

```

**成功的標誌：**
```

📅 環境: development
```

**可能會看到的錯誤（這是正常的）：**
```

```

### 5.3 測試伺服器

1. **保持 PowerShell 視窗開啟**（不要關閉）

2. **開啟瀏覽器**（Chrome、Edge 都可以）

3. **前往**：http://localhost:8080/health

4. **應該會看到**：
   ```json
   {"status":"ok","timestamp":"2026-01-26T..."}
   ```

如果看到這個，恭喜！🎉 **你的開發環境已經設定完成了！**

### 5.4 停止伺服器

當你想停止伺服器時：
- 回到 PowerShell 視窗
- 按 `Ctrl + C`
- 輸入 `Y` 確認（如果需要）

## 🆘 遇到問題？

### 問題 1：找不到 npm 命令

**解決方法：**
1. 重新安裝 Node.js（確保勾選「Add to PATH」）
2. 重新啟動 PowerShell
3. 如果還是不行，重新啟動電腦

### 問題 2：npm install 很慢

**解決方法：**
使用淘寶鏡像（大陸用戶）：
```powershell
npm config set registry https://registry.npmmirror.com
npm install
```

### 問題 3：Port 8080 已被使用

**解決方法：**
1. 修改 `.env` 檔案中的 `PORT=8081`
2. 重新啟動伺服器
3. 瀏覽器訪問：http://localhost:8081/health

### 問題 4：找不到 .env 檔案

**解決方法：**
1. 在檔案總管中，點擊「檢視」→ 勾選「隱藏的項目」
2. 或在 PowerShell 中執行：
   ```powershell
   Get-ChildItem -Force
   ```
